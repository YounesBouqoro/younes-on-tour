## Installation

1. Alte statische Dateien durch diese Dateien ersetzen.
2. Deine echten Werte in `firebase-config.js` eintragen.
3. In `firestore.rules` deine Admin-E-Mail einsetzen.
4. Firestore-Regeln in Firebase veröffentlichen.
5. GitHub Pages: Deploy from branch → main → root.

## Meilensteine

Beim ersten Start erscheinen automatisch:

- Venlo erledigt
- Xanten geplant
- Amsterdam geplant
- Paris geplant
- Gardasee geplant

Sobald du im Adminbereich eigene Meilensteine speicherst, werden diese Standardwerte ersetzt.
